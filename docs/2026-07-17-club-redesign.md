# 运动俱乐部风 — 设计规格

## 配色
| Token | 值 | 用途 |
|--------|-----|------|
| 深绿 | #0a1a0f | 顶栏、底栏、Tab栏 |
| 荧光绿 | #00e85c | 主强调、按钮、数字 |
| 金色 | #c8963e | 点缀、签出标记 |
| 卡片白 | #fff | 卡片底色 |
| 页面底 | #f0f2ee | 内容区 |

## 字体
Barlow Condensed（标题数字）+ Plus Jakarta Sans（正文）+ Noto Sans TC（中文）

## 布局
- 顶栏：深绿底，白色队名大字居中，日期小字，☰左[批量]右
- Tab栏：深绿底，激活态荧光绿胶囊，间隔金色分隔线
- 卡片：白底 16px 圆角，阴影。未签到金色顶边，已签到荧光绿填充，已签退浅金底
- 底栏：深绿底，数字 2xl 荧光绿/白，标签白字不透明

## 页面
4 个路由全部改：/、/kiosk/[teamId]、/view/[teamId]、/admin
